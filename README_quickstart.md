
# Inicio rapido (sin tecnicismos)

## Windows
1. Doble clic en `Instalar_dep.bat`
2. Doble clic en `Generar_tests.bat`
3. Veras los tests en `build\tests\`

## Ubuntu
```bash
./instalar.sh
./generar_tests.sh
```
Los tests aparecen en `build/tests/`.

Parametros por defecto: PDF desactivado, 10 tests, no repetir.
